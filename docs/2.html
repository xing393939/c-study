1.编程环境搭建

编程环境搭建

  1. 安装CLion 2019
  2. 配置编译环境有三种方法：Cygwin、Mingw、Remote host
    1. Cygwin的安装方法，见链接
    2. Mingw的安装方法暂无
    3. Remote host即远程linux环境，如centos：
      • yum install g++
      • yum install gcc-c++
      • yum install cmake
      • yum install gdb
  3. 其中 MinGW 是 Cygwin 下的软件包，具体区别见Cygwin 和MinGW 的区别与联系是怎样的
